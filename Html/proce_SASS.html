Preprocesadores CSS: SASS/SCSS

Temario

Exploramos las principales características de SASS/SCSS: mixins, variables, anidamiento y funciones.

Metodología

Aplicamos una metodología modular con archivos SCSS separados por responsabilidad (variables, mixins, etc.) y usamos Bootstrap como base de estilos.

Características

  • Mixins: Reutilización de bloques de estilos.
  • Variables: Control de colores y fuentes de forma global.
  • Anidamiento: Organización clara de selectores.
  • Funciones: Lógica personalizada dentro del CSS.

Usos comunes

Temas de UI personalizados, frameworks CSS, grandes proyectos web que requieren escalabilidad y mantenibilidad.

Ejemplos

Este sitio usa mixins para bordes redondeados, funciones para espaciados y variables para mantener consistencia visual.

Preprocesadores CSS: SASS/SCSS

Temario

Exploramos las principales características de SASS/SCSS: mixins, variables, anidamiento y funciones.

Metodología

Aplicamos una metodología modular con archivos SCSS separados por responsabilidad (variables, mixins, etc.) y usamos Bootstrap como base de estilos.

Características

  • Mixins: Reutilización de bloques de estilos.
  • Variables: Control de colores y fuentes de forma global.
  • Anidamiento: Organización clara de selectores.
  • Funciones: Lógica personalizada dentro del CSS.

Usos comunes

Temas de UI personalizados, frameworks CSS, grandes proyectos web que requieren escalabilidad y mantenibilidad.

Ejemplos

Este sitio usa mixins para bordes redondeados, funciones para espaciados y variables para mantener consistencia visual.

Variables en SCSS

Las variables en SCSS nos permiten reutilizar valores como colores o tamaños a lo largo del proyecto.

SCSS:
$primary-color: #3498db;
    $padding-base: 1rem;
    
    .button {
      background-color: $primary-color;
      padding: $padding-base;
    }
CSS generado:
.button {
      background-color: #3498db;
      padding: 1rem;
    }
Mixins en SCSS

Los mixins permiten reutilizar bloques de código CSS con parámetros si se desea.

SCSS:
@mixin flex-center {
      display: flex;
      justify-content: center;
      align-items: center;
    }
    
    .box {
      @include flex-center;
      height: 100px;
    }
CSS generado:
.box {
      display: flex;
      justify-content: center;
      align-items: center;
      height: 100px;
    }
Anidamiento en SCSS

Permite escribir selectores de manera jerárquica como reflejo de la estructura HTML.

SCSS:
.navbar {
      background-color: #222;
    
      ul {
        list-style: none;
    
        li {
          display: inline-block;
    
          a {
            color: white;
            text-decoration: none;
          }
        }
      }
    }
CSS generado:
.navbar {
      background-color: #222;
    }
    .navbar ul {
      list-style: none;
    }
    .navbar ul li {
      display: inline-block;
    }
    .navbar ul li a {
      color: white;
      text-decoration: none;
    }
Funciones en SCSS

Las funciones nos permiten crear utilidades personalizadas para operaciones como unidades, colores, etc.

SCSS:
@function rem($px) {
      @return $px / 16 * 1rem;
    }
    
    .text {
      font-size: rem(24); // 1.5rem
    }
CSS generado:
.text {
      font-size: 1.5rem;
    }
